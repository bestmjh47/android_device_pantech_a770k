android_device_pantech_a770k
============================

CyanogenMod10 device tree for IM-A770K (SKY Vega Racer) - Jellybean Branch

Working:
* RIL
* Graphics
* Sound
* Wifi
* Bluetooth
* Sensors
* GPS
* Camera
* Video
* Tethering

Known issues:
* Can not receive MMS 
* Bluetooth SCO is not working
* HDMI is not tested
* Properiatary softwares are not working ( DMB, etc...)
